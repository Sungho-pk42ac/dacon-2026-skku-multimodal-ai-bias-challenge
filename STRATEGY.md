# 🔥 전략 — 가장 공격적인 돌파 작전 (대상 직격)

> 2026 성균관대학교 멀티모달 AI Bias 챌린지 / 솔로 / 클라우드(H100)
> 목표: **대상(총장상).** 방식: **압도적으로 강하고 일반화되는 모델로 정량 70점을 먹는다.**

---

## 0. 전제 (확정 사실)

- **채점 = Balanced Accuracy** = (Acc_ambiguous + Acc_disambiguated) / 2. 3지선다, 정답 인덱스 0/1/2.
- **Public 포화**: 9팀이 이미 만점 1.0, 15위 컷 ~0.998. → **Public 점수는 신호가 아니다(무의미).**
- **최종 수상 = 2차 100점**: 정량 70 (1차 Private 환산 **35** + Hidden 데이터셋 **35**) + 정성 30 (Novelty 20 + 완성도 10).
- **핵심 통찰**: 정량 70점 = 사실상 "내 모델 정확도"다. Hidden은 *내 코드를 더 많은 데이터에 돌린 것*뿐 → **모델이 일반화만 잘하면 70점은 그대로 내 것.** 발표 30은 따라온다.

## 1. 작전 기조

**시간의 ~90%를 "모델 성능"에 박는다.** 발표·재현성은 "감점 안 당할 위생(hygiene)"으로만 처리.
현실론으로 움츠리지 않는다. **이기는 단 하나의 길 = 가장 강하고 일반화 잘 되는 모델.**

---

## 2. 무기 0 — 자체 검증셋 (모든 공격의 조준경, 선행 필수)

Public이 죽었으므로 **내 모델이 진짜 강한지 잴 유일한 수단 = 자작 holdout.**
- ambiguous/disambiguated **균형** 구성, 학습셋과 분리.
- **단일 북극성 지표 = 자체 Balanced Accuracy** (ambiguous/disambiguated 각각도 따로 추적).
- 모든 실험 선택(모델·데이터·하이퍼파라미터·앙상블)은 이 숫자로만 결정.
- ⚠️ 이게 없으면 깜깜이 → 공격 자체가 불가능. **가장 먼저 만든다.**

## 3. 공격 수단 (병렬로 전부 던진다)

1. **체급 최대화** — A6000 48GB·0.5초 안에 들어가는 한 **최강 VLM**.
   - 7B(Qwen2.5-VL)로 베이스라인 → 즉시 **32B급(AWQ/GPTQ 4bit로 48GB에 욱여넣기)**, InternVL 상위 체급 도전.
   - 들어가기만 하면 큰 모델이 일반화에 유리.
2. **데이터 물량·다양성 폭격** — 대회 포맷 기반 합성 **수만 건**.
   - 편향 축 전수 커버(성별·인종/민족·나이·직업·장애·종교·외모…).
   - 추론(`<think>` 증거/편향 점검) 라벨 + GPT 검증 필터로 품질 관리.
   - **데이터가 정량 70점의 핵심.** 양과 다양성에 돈·시간 아끼지 않는다.
3. **풀 파이프라인 끝까지** — 추론형 SFT → **GRPO**(보상=정답+올바른 기권)로 일반화 쥐어짜기.
   - 단, **1차 자체 BA 확보 후** GRPO 도전(과잉공학 방지).
4. **앙상블 + LLM 중재** — 서로 다른 VLM 2~3개 파인튜닝 → LLM이 최종 1개 *생성*(규칙 충족).
   - Hidden 일반화에 가장 강한 한 수.
5. **테스트타임 공격(0.5초·규칙 내)** — self-consistency 다중샘플 + 멀티프롬프트 + 이미지 TTA → LLM 종합.
6. **Hidden 직격 일반화** — Public 과적합 금지, 강한 augmentation·분포 다양화로 "안 본 데이터"에 강하게.
7. **빠른 반복** — H100로 실험 다발 → 자체 검증셋으로 **무자비하게 선택**.

## 6. 자율 반복 학습 루프 (무자비 반복 — 핵심 엔진)

아이디어: **"목표 점수 넘을 때까지 학습을 반복."** → 채택. 단 **타깃을 교정해야 작동한다.**

### ❌ Public 0.99958을 타깃으로 루프 = 안 됨
- Public 포화(9팀 1.0)라 **신호가 아님** — 넘어도 그냥 9~10등.
- 제출 **1일 5회 제한** → 빠른 자동 루프 불가.
- Public 과적합은 **Private/Hidden(정량 70점)을 오히려 해침.**

### ✅ 진짜 루프 타깃 = 자체 검증셋 Balanced Accuracy
무제한·무료·즉시 측정 가능. 이걸로 무한 반복한다.
```
while (자체BA < 목표) and (예산 남음):
    데이터/모델/하이퍼파라미터 조정
    → 학습(SFT/GRPO) → 자체 holdout 평가(BA)
    → 최고 갱신 시 체크포인트 보관 + 로그
주기적으로(≤5/일) "최고 모델만" 리더보드 제출 → Public/Private 정합 확인
```
- **정지조건**: 자체BA 목표 도달 OR N라운드 무개선(정체) OR **예산 상한 도달**.
- **가드레일**: H100 시간/비용 **캡 필수**(무인 루프가 돈을 태움). 라운드마다 보고.
- **구현**: autoresearch/ralph류 루프 + 자체BA 평가자. SSH 받으면 Claude가 라운드별 구동·보고(예산 캡, 제출은 매번 승인).

> 요약: **"Public 넘을 때까지"가 아니라 "자체 BA가 천장 칠 때까지" 무자비하게 반복.** 그게 Private+Hidden을 먹는 길.

## 7. 정직한 리스크 (숨기지 않음)

- **분포 불일치**: 합성 데이터가 실제 test 이미지 도메인과 어긋나면 Hidden에서 무너짐 → 실제 데이터 받아 분포 맞추는 게 급선무.
- **체급 vs 0.5초**: 32B는 양자화해도 A6000 0.5초가 빠듯 → 반드시 A6000 기준 실측.
- **GRPO 비용/시간**: 솔로 D-15에 과투자 위험 → 1차 점수 확보 후에만.
- **비용**: H100 + GPT 합성 데이터 → 예산 상한 정하고 진행.
