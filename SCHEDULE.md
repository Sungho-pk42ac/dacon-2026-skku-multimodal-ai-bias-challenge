## ⚖️ 평가 방식 (공식 규칙, 2026-06-15 확정)
- **1차 = Private Score 100%** → 상위 **15팀(+예비 5)** 선발.
  - Public = 전체 테스트의 ~60% 샘플 / **Private = 나머지 ~40%** (6.29까지 비공개).
  - ⚠️ **Public 만점 ≠ 2차 진출 보장.** 진출은 *Private*가 결정 → **북극성 = 일반화(robustness)**.
  - 최종 채점 파일은 *직접 선택 가능* → Public 최고점이 아니라 **가장 견고한 모델** 선택.
  - 제출 1일 5회, 매 제출 시 최고순위 파일 자동 선택.
- **2차 = Hidden셋 평가 + 발표 + 코드 심사** (운영진 내부평가).
  - Hidden 정량식: `배점 × (팀점수 / 최고팀점수)^N` (N=1~5 비공개) → **상대평가·상위편향**(1등과 격차 크면 급감).
