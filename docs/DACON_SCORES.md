## 텍스트집중(중립 이미지) 레짐 = 1.0 (확인 경위)
- 위 표에서 **v8b 텍스트집중 = 1.0**이 직접 검증됨.
- **v6 텍스트집중 = 1.0**: v4~v6 텍스트집중(placeholder) 제출이 모두 1.0이었음(제출자 확인 + 제출 이력). v6와 v8b의 예측은 실이미지에서 **99.39% 일치(8500중 52개차)**이므로(§5.3), v6 텍스트집중도 v8b와 동일하게 1.0. 즉 §5.3의 "v6 텍스트집중 1.0"은 v8b-텍스트집중 1.0(검증)과 v6≈v8b 동등성에 근거한다.
- **주의(정직성)**: 이 1.0은 DACON **Public 리더보드** 채점값이다. 1차 top-15는 비공개 **Private(40%)**가 결정하므로, Public 1.0이 곧 진출을 보장하지 않는다(본 문서·논문 §9에 명시). 또한 텍스트집중이 실이미지보다 높은 것은 시각 채널 본질적 무정보가 아니라 *본 파이프라인(중립이미지 학습)*의 분포 특성일 수 있다(논문 §6.3 경쟁가설).
