## 3. 핵심 발견
1. **파인튜닝이 일반화를 해치지 않았다(오히려 개선).** 완전 미관측 ood2에서 v6(0.7783)·v8b(0.775) > base(0.705).
   협소 BBQ 파인튜닝이 일반 추론을 붕괴시킬 거란 우려를 독립셋이 반증. (v7의 "파국적 망각"은 콜드스타트 SFT 고유 문제였음을 재확인 — v6/v8b는 GRPO-only/강건성GRPO라 무사.)
2. **v6 ≥ v8b (독립셋에서도).** 정확도 v6 미세우위(0.7783 vs 0.775, MMLU 0.7667 vs 0.7533에서 갈림),
   image-invariance v6 우위(0.975 vs 0.9717). shuffle만 v8b가 +0.005(노이즈). **v8b가 v6를 "명확히 능가"하는 항목 없음.**
3. **base의 한계가 형식+상식에서 드러남.** fmt유효 0.0(단일토큰 미준수), HellaSwag 0.4267(상식 약점). 파인튜닝이 형식·상식 모두 보강.
4. **속도** 모두 0.08~0.12s/샘플 → DACON 0.5s/샘플 제한·Hidden 13분 한도 여유 충족.

## 4. 결론
v6는 **일반화(v8eval OOD 최고 0.855, ood2 독립 최고 0.7783)·편향안전(balanced 1.0, s_AMB 0)·image-invariance(0.975)·속도** 전반에서
best 또는 tied이며, v8a·v8b 어느 쪽도 v6를 명확히 능가하지 못한다.
→ **결정규칙(혼조 시 v6, 후보가 v6를 "명확히" 능가할 때만 교체) 적용 결과: v6 유지.** 상세 결정은 `FINAL_MODEL_SELECTION.md`.
