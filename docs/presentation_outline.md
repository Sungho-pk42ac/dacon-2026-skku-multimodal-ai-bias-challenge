# 솔루션 발표자료 서사 골격 (15분 PDF)

> 규칙: 15분 분량 자유양식 **PDF**(PPT 불가). 결과 확정 후 `/ppt-design`으로 슬라이드덱(HTML→PDF) 제작.
> 핵심 서사 = **"편향을 억제하는 기권 + 정직한 검증 + 속도 컴플라이언스"** (숫자는 결과 확정 후 기입)

---

## 1. 문제 정의 (1~2분)
- 멀티모달 BBQ: 이미지+텍스트 사회적 맥락 질의응답. 근거 충분→정답, 부족→**기권("cannot be determined")**.
- 채점 = **Balanced Accuracy = (ambiguous Acc + disambiguated Acc)/2** → **기권 품질이 점수의 절반**.
- 제약: 학습데이터 미제공(예시 1개), 추론 0.5초/샘플·A6000 오프라인, 모델 6/1 이전.

## 2. 핵심 통찰 (2분)
- **근거는 텍스트(context)에 있고, 이미지는 사회적 단서 추측을 유도하는 "편향 함정"** — 실제 데이터 분석으로 확인.
- ⇒ 승부처 = "언제 답하고 언제 모름을 고를지" (abstention) + 편향 억제.

## 3. 데이터 전략 (2~3분)
- 학습데이터 자체 구성: **공개 BBQ(walledai/BBQ)** 로 ambiguous/disambiguated 균형 학습.
- **규칙 안전선**: 평가셋 형식·문항 모방 ❌ (Data Leakage). 일반 벤치마크로 *스킬*만 학습.
- 🔑 **차별화: val 누출을 직접 발견·제거** — sha1 sample_id(SSOT)로 train/val 완전분리 → **정직한 BA**.
  - (멀티에이전트 검증 회의에서 기존 검증셋의 누출을 적발한 사례 → 방법론적 엄밀성 어필)

## 4. 모델·방법 (3~4분)
- 베이스 **Qwen3-VL-8B-Instruct** (6/1 이전·Apache-2.0).
- **2단계**: SFT(단일토큰 직답, nothink) → (선택) **GRPO**로 Balanced Accuracy 직접 최적화(보상=정답·기권 일치).
- **속도 컴플라이언스**: 단일토큰(`max_new_tokens=4`) + 이미지토큰 축소 → 0.5초/샘플 달성.
  - 대비: 추론형 `<think>` 모델은 1.65초(규칙 위반) → 단일토큰으로 해결한 과정 제시.

## 5. 정량 결과 (2분) — *결과 확정 후*
- 자체검증 BA: base → SFT(v4) → GRPO(v5) 표 + ambiguous/disambiguated 분해.
- 속도 실측(A6000): ___ s/sample (PASS).
- 최종 선택 근거: 정직한 BA + 0.5초 + 일반화.

## 6. 정성 분석 (1~2분)
- 편향 사례: 사회집단 단서가 있어도 근거 없으면 기권하는 예시(이미지 함정 회피).
- 실패/한계: 멀티모달 라벨 검증셋 부재 → 이미지 기여도 정밀측정 한계(정직하게).

## 7. 규칙 준수·재현성 (1분)
- 오프라인·모델날짜·학습추론분리·무누수·LLM생성답 전부 충족(체크리스트).
- seed 고정·requirements 명시 → Private 복원 가능.

## 8. 마무리 (1분)
- **메시지: "더 많은 기법이 아니라, 정직한 검증 + 지표를 직접 최적화한 적은 수의 레버가 일반화를 만든다."**
