정성평가: **솔루션 우수성(Novelty) 20점 + 완성도 10점.**

## 발표 서사 (Novelty 축)

1. **문제 재정의**: Balanced Accuracy = 정확도 경쟁이 아니라 "언제 기권할지"의 문제.
2. **기권 메커니즘**: ambiguous일 때 '알 수 없음'을 고르도록 한 데이터 설계 + 구조화 출력(ambiguous 플래그).
3. **편향 억제 근거**: 사회적 단서(성별·인종) 기반 추측을 줄였음을 정성 사례로 제시.
4. **오프라인 멀티-전문가 아키텍처**: VLM 백본 + (LangGraph 기권 게이트) + (multi-LoRA 답변가/기권탐지가) + LLM 중재 — 전부 오프라인·LLM 생성(규칙 충족).
5. **정량 결과**: ambiguous/disambiguated 정확도 각각 + Balanced Accuracy 개선 곡선.
6. **재현성**: 학습/추론 분리, seed 고정, 환경 명시.
